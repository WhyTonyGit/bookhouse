Требуется разработать информационную систему для учёта библиотечного фонда и обслуживания читателей.

### Хранимые данные
Система должна хранить:
- сведения о книгах: название, автор(ы), издательство, год выпуска, количество страниц, число иллюстраций, стоимость;
- сведения о местах хранения: филиал библиотеки / книгохранилище, в котором находится книга;
- количество экземпляров каждой книги, доступных в библиотеке и в конкретном филиале;
- статистику выдачи: сколько студентов получали конкретную книгу;
- перечень факультетов, где книга используется в учебном процессе (с привязкой к филиалу).

### Требуемый функционал:
1. Для заданного филиала определить количество экземпляров указанной книги, находящихся в этом филиале.
2. Для заданной книги в выбранном филиале вычислить, на скольких факультетах она используется, и вывести список названий этих факультетов.
3. Реализовать добавление и редактирование информации о книгах.
4. Реализовать добавление и редактирование информации о филиалах/книгохранилищах.
5. Предусмотреть механизм хуков/обработчиков, которые срабатывают при пользовательских ошибках и исключительных ситуациях.
